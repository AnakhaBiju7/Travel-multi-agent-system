# Travel Multi-Agent System
A **Level-4 Autonomous AI MAS** for hotel selection and monitoring.  
Built for research and experimentation with multi-agent debate, critic evaluation, and autonomous decision-making.

---

## Features
- Multi-agent hotel selection
- Critic agent for opinion evaluation
- Final synthesizer for optimal decision
- Continuous price monitoring
- Memory of past decisions and price changes
- Extensible for other travel domains (flights, packages, etc.)
